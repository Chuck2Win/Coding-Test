7. count - `O(N)`

   - 작은 범위의 정수 정렬에 유용. 
   - 숫자의 발생 횟수를 계산하는 `누적 카운트` 사용
   - 숫자 간 간격이 작아야 좋음

   ```{python}
   from collections import defaultdict
   def count_sort(a):
       # a - N개의 수
       count = defaultdict(list)
       for i in a:
           count[i].append(i) # O(N)
       sorted_a = []
       for i in range(min(count),max(count)+1): # O(K)
           sorted_a.extend(count[i]) #O(N)
       # so O(N+K)
       return sorted_a
   ```
